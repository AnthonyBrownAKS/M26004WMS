# M26004WMS

## 项目简介
本项目包括前端操作页面，后端业务逻辑，数据库管理等技术
通过本项目的设计与实现，旨在实现以下功能： 
（1） 入库操作与数据库数据同步更新
（2） 出库操作与数据库数据同步更新 
（3） 堆垛机通信传递数据信息
（4） 基本增删改查数据库交互功能 
（5） 前端操作页面方便人员操作

## 技术栈
- Vue3
- SpringBoot
- Mysql
- 标准WMS+WCS模式

## 功能
1) 物料管理模块：提供增删改查物料信息功能，用户可以修改查看物料名称，物料编号，容器编号，库位编号，入库时间，出库时间。
2) 容器管理模块：提供增删改查容器信息功能，用户可以修改查看容器编号、客商代码，物料编号，物料数量，库位编号。
3) 库位管理模块：提供改查库位信息功能，用户可以查看查看库位编号、库位分区、容器编号。
4) 出入库管理模块：提供物料出入库业务操作接口，用户可通过该模块发起入库或出库请求，并由系统生成对应任务。
5) 设备控制模块（WCS层）：负责与底层自动化设备（堆垛机PLC）进行通信，实现任务的具体执行控制。
6) 任务调度模块：负责对出入库任务进行统一管理与调度，实现任务的执行控制、状态流转及资源分配。
